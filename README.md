# Blokus Project

This repository contains the Blokus-like block puzzle game.

Run locally:

1. Install dependencies:

```bash
npm install
```

2. Run dev server (if using Vite):

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
# then serve the `dist/` directory
```

GitHub Pages deployment is configured with a GitHub Actions workflow that builds and publishes `./dist` using `peaceiris/actions-gh-pages`.
